# 머신러닝

머신러닝은 ai를 달성하기 위한 수단. 
머신러닝의 방법 중 유명한 건 지도학습, 비지도학습.

지도학습: 라벨링 (O, X)해서 제공함. 우리가 준 라벨을 토대로 기계는 판단.
제공하는 데이터에 정답이 있고, 정답 있는 데이터로 트레이닝 시켜서 정답예측 모델을 만드는 것.
기계는 라벨이 붙은 data를 토대로 학습한다.
인간이 data를 라벨링 해서 제공. data를 분류해서 제공.

비지도학습: 
data에 대한 라벨이나 설명없이 그냥 data만 줌. ex 핫도그 사진만 엄청..
알아서 컴퓨터가 군집화 하며 비슷한 것끼리 묶음.

강화학습:  


* tensoflow: AI를 위한 가장 유명한 프레임워크. 자바스크립트와 파이썬으로 만들어져있음.
머신러닝 위해선 파이썬과 자스가 필요하다. 파이썬은 이미 엄청 거대하고 자스는 이제 시작하는 정도.




#딥러닝

딥러닝은 머신러닝을 달성하기 위한 수단.  머신러닝에는 인간이 가이드를 주지만 딥러닝에는 가이드를 주지 않고 데이터만 엄청 많이 집어넣는다.?

딥러닝: neural network 이용
아주 길고 인텐시브한 프로세스, 프로세싱 파워가 많이 필요함.

딥러닝은 뉴럴 네트워크를 이용해 머신러닝을 진행하는 것.

input layer -  hidden layer   -  output layer

이 사이의 넷월ㅋ  = neural network
hidden layer가 복잡할 수록 deep nueral network.  레이어가 여러개일수록 정교하다.

cf. perceptron


알아서 특징을 추출함 feature extraction.  전통머신러닝은 사람이 가이드를 줬는데, 딥러닝은 필요 없다는 거 .

손실함수 loss function: 모델이 정확한지 오차 계산
활성함수 activation function : 비선형적이고 복잡한 예측이 가능해짐.





#표

row : instance(개체), observed value(관측치), record, example, case
column : feature(특성), attribute(속성), variable(변수) ..



#독립변수와 종속변수

표에서 column : 변수.

독립변수: 원인이 되는 열
종속변수: 결과가 되는 열

표를 보면 : 이 표의 특성 중 서로 상관있는 특성이 무엇인지를 우선적으로 탐색해야 한다. 즉 표 내의 열 간의 상관관계를 찾기.


#머신러닝의 분류

기계학습 : 지도학습(분류+회귀)  +  강화학습 +   비지도학습(군집화+ 변환+ 연관)

----------

지도학습 supervised learning  (지도: 기계를 가르치다)
data로 기계를 가르치는 것.  정답이 있는 문제를 해결.

비지도학습 - 지도학습에 속하지 않는 방법 unsupervised learning 
기계에게 data에 대한 통찰력을 부여하는 것. 무언가에 대한 관찰을 통해 새로운 의미나 관계를 밝혀냄.

강화학습 reinforcement learning
학습을 통해 능력을 향상시킨다는 점에서 지도학습과 유사하나,
지도학습이 정답을 알려주지만 강화학습은 그렇지 않다. 경험을 통해 더 좋은 답을 찾아감.

--------



#지도학습

과거의 data로부터 학습해서 결과를 예측하는 것에 주로 사용됨.
과거의 data를 독립변수와 종속변수로 나누고, 그 관계를 컴퓨터에게 학습시키면 컴퓨터는 그 관계를 설명할 수 있는 공식을 만들어 냄.
이때의 공식이 model.
이렇게 학습시키면 결과를 모르는 독립변수만을 제공했을 때 그 결과를 예측해낸다.


지도학습은 분류와 회귀로 나뉜다.

분류 classification  :예측하려는 종속변수(결과)가 문자일 때.(숫자가 아닐 때.) ex: 합격 여부, 와인의 등급 등..
회귀 regression   : 예측하고 싶은 종속변수가 숫자일 때 회귀 방식 사용.


------

#비지도 학습 


군집화 clustering : 비슷한 것을 찾아서 그룹을 만드는 것.
 - 지도학습의 분류와의 차이는?
어떤 대상들을 구분해서 그룹을 만드는 게 군집화, 어떤 대상이 어떤 그룹에 속하는지를 판단하는 게 분류.
비슷한 행을 그룹핑하는 것.

변환 transform

연관 association rule learning  연관 규칙 학습. 일명, 장바구니 학습.
관련이 있는 열을 찾는 것.  cf. 군집화는 행의 관계를 봐서 행을 그룹핑하고, 연관은 열들의 관계를 봐서 관련있는 열들을 찾는다.


-----

#지도학습과 비지도학습 비교
 
비지도 학습 :  탐험적. data의 성격을 파악하는 것. 애초에 독립/종속변수 구분 x.
            군집화, 연관 모두 비슷한 것을 모으는 작업이라는 점에서 공통적. 데이터를 정리해서 표에 담긴 데이터의 성격을 파악하는 게 목적.
지도 학습:  역사적. 과거의 원인, 결과를 바탕으로 결과 모르는 원인 발생했을 때 그것의 결과를 추측하는 게 목적.
        그래서 독립변수와 종속변수 구분이 필요.


-----

머신러닝(기계학습) 
=  지도학습(분류, 회귀)  + 비지도학습 (군집화, 변환, 연관)  + 강화학습     :으로 분류된댔지
그럼 이제 강화학습을 보자


#강화학습   reinforcement learning

지도학습이 배움을 통해 실력 키우면
강화학습은 일단 하는 경험을 통해 실력 키운다.
과거의 data 없고, 보상을 되돌려주는 환경은 있을 때.

환경. 에이전트. 정책(판단). 보상. 상태 

상태에 따라 더 많은 보상을 받을 수 있는 행동을 할 수 있는 에이전트를 만드는 정책.
